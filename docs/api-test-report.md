## 발견된 이슈 / 주의사항

| 항목 | 내용 | 상태 |
|---|---|---|
| 이메일 대소문자 | 이메일은 대소문자 구분 저장 (`user@test.com` ≠ `USER@TEST.COM`) | 동작 확인 (의도된 동작) |
| 팀 중복 가입 | 한 사용자가 다른 팀으로 합류 시 `team_id`가 덮어쓰기됨 (한 사람 = 한 팀) | 스펙 의도대로 동작 |
| DELETE 응답 | 204 No Content — body 없음 | 정상 |
